# 更新日志

本文档记录项目的重要变更。发布流程会根据当前 tag 读取对应版本段落作为 GitHub Release 内容。

## [v0.1.0] - 2026-05-09

### 新增

- 初始版本：基于 Tauri 2 的桌面应用，支持将笔记批量导出为 Markdown。
- 前端采用 React 19、TypeScript、Vite，后端采用 Rust 和 Tauri 命令层。
- 新增完整同步流程：Token 校验、分页拉取笔记、Markdown 导出、本地索引更新、进度事件推送和取消同步。
- 新增应用设置持久化：Token、默认导出目录、分页大小、上次同步模式、导出目录结构、文件名规则、同步后打开目录和同步提示。
- 新增同步概览和历史记录持久化，展示上次同步信息、最近导出记录和近期失败项。
- 新增首次使用引导，包含欢迎、Token 配置、导出目录配置和完成步骤。
- 新增导出目录结构配置，支持平铺、按月份、按标签组织 Markdown 文件。
- 新增文件名规则配置，支持标题加 ID、日期加标题加 ID。
- 新增同步页状态卡、模式说明、Token 缺失提示、最近导出展示和失败项展示。
- 新增设置页字段级自动保存；Token 保持独立保存按钮，避免敏感信息误保存。
- 新增 macOS、Windows、Linux 跨平台界面适配，包括标题栏间距、字体栈和平台样式。
- 新增应用图标和品牌资源，用于桌面应用打包。

### 优化

- 重构同步页布局和交互，让同步状态、历史摘要和同步后快捷操作更清晰。
- 更新应用配置和打包元数据，接入应用图标资源。
- 优化 CI/CD 和发布工作流配置，支持多平台构建。

### CI

- 新增 GitHub Release 创建权限配置。
- 新增 macOS、Windows、Linux 发布构建任务。
- 新增平台构建目标：macOS Universal、Windows x64、Linux x64。

### 文档
