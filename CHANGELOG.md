# 更新日志

本文档记录项目的重要变更。

## [v0.2.1] - 2026-05-10

### 新增

- 按知识库分组导出：新增按「知识库」目录结构，按笔记所属知识库创建子目录
- 仅标题文件名：新增「仅标题」命名规则，文件名更简洁，不含 ID 后缀
- 导出文件 YAML 元数据新增 `topics` 字段，记录知识库归属
- 自动更新：应用启动后自动检测新版本，后台静默下载，一键重启即可更新
- 设置页新增「关于」区域，可查看版本号和手动检查更新
- 标题栏更新提示：新版本下载完成后，标题栏显示提示点，点击即可重启更新
- 同步日志持久化：同步日志保存到本地文件，关闭应用后仍可查看历史日志

### 改进

- 诊断日志改为可折叠面板，自动展开、可手动收起，不再限制显示条数
- 同步失败时显示错误提示横幅，可关闭
- 进度条优化：未知总数时显示滑动动画，不再显示无意义的进度比例
- 增量同步日志显示上次同步时间，更直观地了解同步范围
- 「打开导出目录」按钮移至操作区，更方便访问
- 导出默认值改为「按知识库」+「仅标题」，更符合常见使用习惯

## [v0.1.0] - 2026-05-09

首个正式版本。

### 核心功能

- 批量同步笔记并导出为 Markdown 文件，支持增量同步和全量同步
- 本地索引记录同步状态，增量模式只处理新增笔记，全量模式可更新远程修改过的笔记
- 同步进度实时展示，支持取消同步
- 同步完成后展示结果摘要和失败项

### 导出配置

- 目录结构：平铺 / 按月份 / 按标签
- 文件命名：标题+ID / 日期+标题+ID
- 每个 Markdown 文件包含 YAML 元数据（标题、ID、标签、时间）

### 应用设置

- Token 密码框输入，支持明文/密文切换
- 字段级自动保存（失焦即保存），Token 保留独立保存按钮

### 首次引导

- 检测未配置状态，自动弹出 4 步引导：欢迎 → Token → 目录 → 完成
- 支持跳过引导

### 跨平台

- 支持 macOS、Windows、Linux
- 平台自适应标题栏间距、字体栈和窗口样式

### 技术栈

- 前端：React 19 + TypeScript + Vite
- 后端：Rust + Tauri 2.x
- CI/CD：GitHub Actions 多平台构建与发布
